# ArUco机器人视觉伺服控制系统

## 当前状态

系统采用模块化架构，当前只激活ArUco跟随功能模块。架构设计支持后续轻松添加新功能模块。

## 运行方法

```bash
python main.py
```

## 控制说明

### 系统控制
- **Q**: 退出程序

### 相机控制（通用）
- **W**: 相机向上
- **S**: 相机向下  
- **A**: 相机向左
- **D**: 相机向右
- **X**: 相机复位

### ArUco跟随控制
- **M**: 启用跟随模式
- **N**: 禁用跟随模式

#### 位置调整
- **I**: Z轴上升（+2cm）
- **K**: Z轴下降（-2cm）
- **J**: Y轴左移（+2cm）
- **L**: Y轴右移（-2cm）
- **R**: X轴前进（+2cm）
- **T**: X轴后退（-2cm）

#### 角度调整
- **U**: Roll角度正向（+3°）
- **O**: Roll角度负向（-3°）
- **Y**: Pitch角度正向（+3°）
- **P**: Pitch角度负向（-3°）
- **G**: Yaw角度正向（+3°）
- **H**: Yaw角度负向（-3°）

#### 预设配置
- **Z**: 追踪tag
- **C**: 下移末端
- **V**: 压下把手

## 系统架构

```
RobotSystem (main.py)
├── ArucoFollowModule (当前活跃)
├── RobotController (机器人控制)
├── ArucoProcessor (ArUco处理)
└── API层 (相机和机器人通信)
```

## 扩展新功能

要添加新功能模块：

1. 继承 `BaseModule` 创建新模块类
2. 在 `main.py` 的 `_initialize_modules()` 中注册
3. 可选：添加模块切换按键

详细说明请参考 `ARCHITECTURE.md`

## 优势

- **模块化**: 功能独立，易于维护
- **可扩展**: 轻松添加新功能
- **统一接口**: 所有模块遵循相同规范
- **资源管理**: 自动处理模块切换和清理

当前系统专注于ArUco跟随功能，为后续功能扩展提供了坚实的架构基础。
